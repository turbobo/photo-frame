> 版本：v1.0.0 · 最后更新：2026-07-02

## 一、产品定位

**一句话价值**：让每一张照片，都值得配上摄影师风格的边框和 EXIF 信息。

**目标用户画像**：
1. **摄影爱好者**（主）：Sony/Canon/Nikon/Fujifilm 相机用户，希望在朋友圈/微博/小红书展示照片时带出机身参数
2. **旅行博主**：iPhone/华为拍摄，希望作品有专业感和辨识度
3. **摄影学生**：需要作业展示带 EXIF 参数

**核心场景**：
- 用户拍完照 → 传到电脑 → 打开 Photo Frame → 拖入照片 → 秒得成品

## 四、边框模板设计

### 4.1 极简 Minimal

- 上下左右等宽白边
- 底部窄条留白（~6% 长边）
- 底部居中一行小字：`型号 · 焦距 光圈 快门 ISO`
- 左下 Logo（可选）

**适用**：Instagram Story, 日常分享

### 4.2 拍立得 Polaroid

- 上左右窄边（~5% 长边）
- 底部大留白（~15% 长边）
- 手写风字体（Snell Roundhand fallback STXingkai）
- 默认显示日期，可自定义（如摄影师签名）

**适用**：日系文艺、小红书

### 4.3 胶片 Film

- 黑色边框 + 白色齿孔（模拟 135 胶卷）
- 左下：`▶ FRAME 0001` 帧号
- 右下：EXIF 参数
- 等宽字体（SF Mono）

**适用**：摄影发烧友

### 4.4 参数栏 EXIF

- 底部深色信息栏（~12% 长边高）
- 左侧：品牌 Logo + 型号（大号） + 镜头（次级）
- 右侧：焦距 · 光圈 · 快门 · ISO 四个 block
- 每个 block：数值粗体 + label 灰色

**适用**：街拍、专业展示（本工具的招牌模板）

### 4.5 社交 Insta

- 白底毛玻璃卡片
- 大圆角（默认 24px，可调 0-80px）
- 阴影效果（可开关）
- 底部一行：Logo + 型号 + 参数

**适用**：小红书、朋友圈

### 4.6 徕卡栏 Leica（NEW）

- 底部黑色窄栏（~6% 长边高），图片尺寸不扩展
- 左侧：红点圆点（#e60012）+ `LEICA` 粗体 + `CAMERA WETZLAR` 小字
- 右侧：等宽字体 EXIF 参数（右对齐）
- 红点直径约为栏高的 22%

**适用**：徕卡用户、高端摄影展示

### 4.7 红点水印 Red-Dot（NEW）

- 悬浮在图片右下角，**不扩画布**（直接覆盖在原图上）
- 半透明黑底圆角块（opacity 35%，提升可读性）
- 左侧红点 + 右侧：型号（大号）+ EXIF 参数（小号）
- 灵感：Leica 官方 Red Dot 风格水印

**适用**：徕卡/索尼用户、极简展示

### 4.8 Dazz 胶卷（NEW）

- 仿 135 胶卷边框：上下齿孔（矩形 + 圆角，米白色）
- 左侧竖排文字：`FUJI SUPERIA 400 ▶ FRAME 0036`

- 右下角日期印字（橙色/黄色，仿旧式相机日期背印）
- 灵感：Dazz Cam、NOMO CAM、OldRoll

**适用**：胶片复古风、Instagram

### 4.9 Instax 真实拍立得（NEW）

- 底部手写签名（Snell Roundhand）+ 右下日期小字
- 灵感：Instax mini / Instax SQUARE

**适用**：旅行记录、手帐风

### 4.10 小红书 XHS（NEW）

- 3:4 比例白底卡片 + 圆角（默认 12px）
- 阴影效果（可开关）
- 顶部：`📕 小红书笔记` 小标签 + 日期
- 中部：图片（圆角剪裁）
- 底部：标题（粗体）+ 描述（EXIF 参数 / 镜头）+ 右下 Logo
- 灵感：小红书图文笔记封面

**适用**：小红书分享、社交媒体

### 4.11 复古纸相框 Vintage（NEW）

- 牛皮纸底（#d4b896）+ 随机噪点纹理（模拟纸张颗粒）

## 五、EXIF 展示规则

### 5.1 显示优先级

按重要性从高到低（每个模板可选择显示部分）：

1. **相机型号**（如 A7M4, EOS R5）— 主体
2. **镜头**（如 FE 24-70mm F2.8 GM）— 副标
3. **焦距**（35mm）— 参数
4. **光圈**（f/2.8）— 参数
5. **快门**（1/250）— 参数
6. **ISO**（ISO400）— 参数
7. **日期**（2026-07-02）— 补充

### 5.2 缺失字段处理

- EXIF 完全缺失（截图/网图）：模板不显示 EXIF 部分，或显示"—"占位
- 部分缺失：只展示存在的字段，用 `·` 分隔

### 5.3 品牌名归一化

用户拍摄的照片 EXIF 中 `Make` 字段可能为：
- `SONY` → 匹配 sony
- `Canon Inc.` → 匹配 canon（去除 `Inc.`）
- `NIKON CORPORATION` → 匹配 nikon
- `FUJIFILM` → 匹配 fujifilm

代码位于 `src/utils/logos.ts::getLogoPath()`。

## 六、导出规则

### 6.1 尺寸预设

| 预设 | 长边 | 用途 |
|------|------|------|
| 原图 | 无缩放 | 打印、归档 |
| 4K   | 3840 | 大屏展示 |
| 2K   | 2048 | 微博、朋友圈 |
| 社交 | 1080 | Instagram、小红书 |

超过预设长边时才降采样，小于则原样输出。

### 6.2 格式

| 格式 | 特性 |
|------|------|
| JPG  | 兼容性最好、体积小（默认 92%） |
| PNG  | 无损、透明通道 |
| WebP | 更小体积、部分平台支持 |

### 6.3 文件名规则

`{原文件名去后缀}-{模板ID}.{扩展名}`

例：`DSC_0123-exif.jpg`

## 八、隐私与安全

- **所有处理在本地**：照片不上传任何服务器
- **EXIF 显示**：GPS 信息仅在用户主动查看时显示，不自动写入水印
- **无 tracking**：不使用 Google Analytics 等第三方 SDK
- **免登录**：无账号系统，即用即走

## 十、指标（成功度量）

| 指标 | 目标 | 说明 |
|------|------|------|
| 上传到预览耗时 | < 1s (JPG), < 2s (RAW) | 用户体感"秒开" |
| 参数调整重绘 | < 50ms | 无卡顿感 |
| 导出成功率 | > 99% | Canvas API 稳定 |
| 首屏加载 | < 300ms | Vite + CDN |
| 单次会话平均导出数 | > 3 张 | 反映产品价值感 |
