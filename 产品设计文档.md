## 一、产品定位

**一句话价值**：让每一张照片，都值得配上摄影师风格的边框和 EXIF 信息。

**目标用户画像**：
1. **摄影爱好者**（主）：Sony/Canon/Nikon/Fujifilm 相机用户，希望在朋友圈/微博/小红书展示照片时带出机身参数
2. **旅行博主**：iPhone/华为拍摄，希望作品有专业感和辨识度
3. **摄影学生**：需要作业展示带 EXIF 参数

**核心场景**：
- 用户拍完照 → 传到电脑 → 打开框影 → 拖入照片 → 秒得成品

### 3.1 整体结构

```
┌──────────────────────────────────────────────────────┐
│ 🖼 框影 · Framelet                [RAW][JPG][PNG][HEIC]│  ← Header 44px
├──────┬─────────────────────────────────┬─────────────┤
│ 模板 │                                 │ 样式|信息|导出│  ← Tabs
│ ─── │                                 │ ─────────── │
│ ▪极简│         Canvas 预览            │  边距 ─●──  │
```

### 3.2 视觉风格

- **整体**：深色主题（slate-950 背景）
- **强调色**：sky-400（品牌蓝）
- **字体**：-apple-system → PingFang SC → Helvetica Neue
- **动画**：所有过渡 0.2s ease-out，卡片悬浮 shadow 提升
- **反馈**：拖拽时边框高亮 + 内部渐变、加载时 sky 色 spinner

### 3.3 交互动效

| 事件 | 反馈 |
|------|------|
| 拖入文件到上传区 | 边框变 sky-400 + 半透明渐变背景 |

## 四、边框模板设计

### 4.1 极简 Minimal

- 上下左右等宽白边
- 底部窄条留白（~6% 长边）
- 底部居中一行小字：`型号 · 焦距 光圈 快门 ISO`
- 左下 Logo（可选）

**适用**：Instagram Story, 日常分享

### 4.2 拍立得 Polaroid

- 上左右窄边（~5% 长边）
- 底部大留白（~15% 长边）
- 手写风字体（Snell Roundhand fallback STXingkai）
- 默认显示日期，可自定义（如摄影师签名）

**适用**：日系文艺、小红书

### 4.3 胶片 Film

- 黑色边框 + 白色齿孔（模拟 135 胶卷）
- 左下：`▶ FRAME 0001` 帧号
- 右下：EXIF 参数
- 等宽字体（SF Mono）

**适用**：摄影发烧友

### 4.4 参数栏 EXIF

- 底部深色信息栏（~12% 长边高）
- 左侧：品牌 Logo + 型号（大号） + 镜头（次级）
- 右侧：焦距 · 光圈 · 快门 · ISO 四个 block
- 每个 block：数值粗体 + label 灰色

**适用**：街拍、专业展示（本工具的招牌模板）

### 4.5 社交 Insta

- 白底毛玻璃卡片
- 大圆角（默认 24px，可调 0-80px）
- 阴影效果（可开关）

**适用**：小红书、朋友圈

### 4.6 徕卡栏 Leica（NEW）

## 五、预设 vs 模板使用指南

### 5.1 概念澄清

| 维度 | 模板（Template） | 预设（Preset） |
|------|----------------|---------------|
| **本质** | 视觉结构（边框/布局/装饰） | 配置快照（模板 + 所有参数） |
| **数量** | 21 个固定 | 5 个官方 + 无限自定义 |
| **修改范围** | 仅切换视觉结构 | 一键应用全部参数 |
| **类比** | 衣服款式（T恤/衬衫） | 整套穿搭（款式+颜色+尺码） |

### 5.2 使用场景对比

#### 场景 1：快速出图（用预设）

**用户画像**：不想调参，只想一键出图

**操作流程**：
```
1. 上传照片
2. 预设下拉 → 选「极简白边」
   ↓ 自动应用：minimal 模板 + 白边 + 宋体 + 完整 EXIF
3. 点击下载
```

**5 个官方预设详解**：

| 预设 | 适用场景 | 配置详情 |
|------|---------|---------|
| **极简白边** | 日常分享、朋友圈 | `minimal` 模板 + 白边 + 宋体 + `{Make} {Model} {FocalLength} {FNumber} {ExposureTime} {ISO}` |
| **专业深色** | 摄影作品展示 | `exif` 模板 + 黑底 + Inter 字体 + `{Model} · {Lens}` |
| **复古胶片** | 胶片风格照片 | `film` 模板 + 黑框齿孔 + JetBrains Mono + `{Make} {Model} {FocalLength} {FNumber}` |
| **社交分享** | 小红书/微博 | `insta` 模板 + 圆角阴影 + Noto Sans + `{Model} \| {DateTime}` |
| **高清打印** | 印刷输出 | `white-border` 模板 + 5% 白边 + 宋体 + 完整 EXIF |

#### 场景 2：精细调参（用模板）

**用户画像**：想微调参数，打造个性化风格

**操作流程**：
```
1. 上传照片
2. 模板区 → 选「徕卡栏」
3. 调整参数：
   - 字号：1.8% → 2.2%
   - 字体：宋体 → 等宽
   - 自定义文字：{Model} · Shot on {DateTime}
4. 满意后 → 预设区 → 「+ 保存」→ 命名「我的徕卡风」
5. 下次直接选「我的徕卡风」预设
```

### 5.3 高级技巧

#### 技巧 1：预设 + 微调 = 新预设

```
1. 选「极简白边」预设
2. 微调：字号 2.5%、字体改手写、自定义文字改「{Location} · {DateTime}」
3. 「+ 保存」→ 命名「旅行专用」
4. 以后上传旅行照片 → 直接选「旅行专用」预设
```

#### 技巧 2：文本变量组合

在「自定义文字」框中，点击下方按钮插入变量：

| 变量 | 输出示例 | 适用场景 |
|------|---------|---------|
| `{Make}` | `Nikon` | 品牌标识 |
| `{Model}` | `Z 5` | 机型标识 |
| `{Lens}` | `24-70mm F2.8` | 镜头信息 |
| `{FocalLength}` | `50mm` | 焦距 |
| `{FNumber}` | `f/2.8` | 光圈 |
| `{ExposureTime}` | `1/500` | 快门 |
| `{ISO}` | `ISO250` | 感光度 |
| `{DateTime}` | `2025-01-15` | 拍摄日期 |
| `{Location}` | `北京·故宫` | 地点（需手动输入） |
| `{Copyright}` | `© 2025` | 版权信息 |

**组合示例**（3 行签名）：
```
{Make} {Model}
{FocalLength} {FNumber} {ExposureTime} {ISO}
{Location} · {DateTime}
```

输出：
```
Nikon Z 5
50mm f/2.8 1/500 ISO250
北京·故宫 · 2025-01-15
```

### 5.4 常见问题

#### Q1：预设和模板冲突怎么办？

**答**：预设 = 模板 + 所有参数。选预设会覆盖当前所有参数（包括模板选择）。

**示例**：
```
当前状态：徕卡栏模板 + 字号 2.2% + 等宽字体
选「极简白边」预设 → 自动切换为 minimal 模板 + 字号 2% + 宋体
```

#### Q2：自定义参数后，预设会丢失吗？

**答**：不会。预设是保存的快照，修改当前参数不影响已保存的预设。

**示例**：
```
保存预设「我的风格」→ 修改字号为 3% → 预设「我的风格」仍是原来的 2.2%
再次选「我的风格」→ 恢复为 2.2%
```

#### Q3：如何删除自定义预设？

**答**：
```
1. 预设下拉 → 选要删除的预设
2. 点击「删除」按钮
3. 确认对话框 → 确定
```

**注意**：官方预设（极简白边/专业深色/复古胶片/社交分享/高清打印）不可删除。

#### Q4：预设会跨设备同步吗？

**答**：不会。预设保存在浏览器 localStorage，仅当前设备可用。

**未来计划**（Roadmap P2）：云同步预设（需登录）。

## 六、EXIF 展示规则

### 5.1 显示优先级

按重要性从高到低（每个模板可选择显示部分）：

1. **相机型号**（如 A7M4, EOS R5）— 主体
2. **镜头**（如 FE 24-70mm F2.8 GM）— 副标
3. **焦距**（35mm）— 参数
4. **光圈**（f/2.8）— 参数
5. **快门**（1/250）— 参数
6. **ISO**（ISO400）— 参数
7. **日期**（2026-07-02）— 补充

### 5.2 缺失字段处理

- EXIF 完全缺失（截图/网图）：模板不显示 EXIF 部分，或显示"—"占位
- 部分缺失：只展示存在的字段，用 `·` 分隔

### 5.3 品牌名归一化

用户拍摄的照片 EXIF 中 `Make` 字段可能为：
- `SONY` → 匹配 sony
- `Canon Inc.` → 匹配 canon（去除 `Inc.`）
- `NIKON CORPORATION` → 匹配 nikon
- `FUJIFILM` → 匹配 fujifilm

代码位于 `src/utils/logos.ts::getLogoPath()`。

## 七、导出规则

### 7.1 尺寸预设

| 预设 | 长边 | 用途 |
|------|------|------|
| 原图 | 无缩放 | 打印、归档 |

### 7.2 格式

|------|------|
| JPG  | 兼容性最好、体积小（默认 92%） |

### 7.3 文件名规则

`{原文件名去后缀}-{模板ID}.{扩展名}`

例：`DSC_0123-exif.jpg`

## 八、批量导出

### 8.1 用户场景

| 用户类型 | 典型规模 | 场景 |
|---------|---------|------|
| 普通用户 | 5-20 张 | 旅行照片一次性加水印 |
| 摄影爱好者 | 20-100 张 | 作品集统一风格 |
| 专业摄影师 | 100-500 张 | 批量交付客户 |

### 8.2 操作流程

```
1. 拖拽多选照片（或点击"批量选择"按钮）
   ↓
2. 选择预设/模板 + 字体 + 自定义文字
   ↓
3. 选择导出格式（JPG/PNG/WebP）+ 质量
   ↓
4. 点击「批量导出 ZIP」
   ↓
5. 全屏进度浮层显示：
   · 进度条（0/100 → 100/100）
   · ETA 剩余时间
   · 当前处理文件名
   · 取消按钮
   ↓
6. 完成后弹窗显示：
   · 成功/失败计数
   · 失败详情（如有）
   · 下载按钮
```

### 8.3 平台适配

| 平台 | 最大批量 | 检测方式 | 说明 |
|------|---------|---------|------|
| Chrome Desktop | 50 张 | `deviceMemory ≥ 4` | 内存充足 |
| iOS Safari | 20 张 | UA 匹配 iPhone/iPad | 内存紧张 |
| Android 旗舰 | 30 张 | `deviceMemory ≥ 4` + Android UA | 内存适中 |
| Android 低端 | 20 张 | `deviceMemory < 4` | 内存紧张 |

**用户提示**：超过平台安全上限时，弹窗提示"建议分批处理，每批 N 张"。

### 8.4 错误处理

| 错误类型 | 处理策略 |
|---------|---------|
| 单张渲染失败 | 记录失败原因，继续处理其他照片 |
| 内存超限 | 弹窗提示"内存不足，建议减少批量大小" |
| 用户取消 | 立即中止，已处理的不保留 |
| ZIP 生成失败 | 弹窗提示 + 重试按钮 |

### 8.5 文件命名规则

- **ZIP 文件名**：`photo-frame-batch-YYYYMMDD-HHMMSS.zip`
- **内部文件名**：`{原文件名}-{模板ID}.{格式后缀}`（与单张导出一致）

### 8.6 批量导出（未来功能，Roadmap P1）

**当前状态**：仅支持单张导出  
**未来计划**：批量 ZIP 导出（20-200 张）

**技术栈**：
- fflate（流式 ZIP，8KB gzipped）
- exifr（EXIF 提取）
- Canvas 2D（渲染）

**关键技术**：
- 逐张渲染 + 立即释放内存（控制峰值 ~220MB）
- 流式 ZIP 写入（不累积 Blob）
- AbortController 支持取消
